# About
ZebraHub is a secure online patient portal empowering rare disease patients and their families to play an active role in ongoing clinical trials. Built for ultra-rare diseases where every participant counts.


📝 Individualized enrollment — registration tailored to each patient's needs  <br>  
📅 Daily check-ins — symptom tracking over time  <br>  
🗓️ Appointment viewer — upcoming appointments at a glance  <br>  
📊 Trial status dashboard — real-time trial progress in one place  <br>
